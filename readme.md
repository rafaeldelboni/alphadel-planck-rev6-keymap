# The AlphaDel Planck Layout

```
  ,-----------------------------------------------------------------------------------.
  | Tab  |   Q  |   W  |   E  |   R  |   T  |   Y  |   U  |   I  |   O  |   P  | Bksp |
  |------+------+------+------+------+-------------+------+------+------+------+------|
  | CEsc |   A  |   S  |   D  |   F  |   G  |   H  |   J  |   K  |   L  |   ;  |  "   |
  |------+------+------+------+------+------|------+------+------+------+------+------|
  | Shift|   Z  |   X  |   C  |   V  |   B  |   N  |   M  |   ,  |   .  |   /  |RLayer|
  |------+------+------+------+------+------+------+------+------+------+------+------|
  |RShift| Ctrl | Alt  | GUI  |Lower | Space|Enter |Raise | Left | Down |  Up  |Right |

```

  |------+------+------+------+------+------|------+------+------+------+------+------|

  |      |      |      |      |      |             |      | Next | Vol- | Vol+ | Play |

## Build environment

1) See [The Complete Newbs Guide To QMK](https://docs.qmk.fm/#/newbs).

2) Clone the QMK repository
```
git clone git@github.com:qmk/qmk_firmware.git
```
3) Clone this repository inside the following folder structure `keyboards/planck/keymaps/alphadel`
```
git clone git@github.com:rafaeldelboni/alphadel-planck-rev6-keymap.git qmk_firmware/keyboards/planck/keymaps/alphadel
```

## Building Instructions

1) from the `qmk_firmware` directory run:
```
$ sudo make planck/rev6:alphadel
```
